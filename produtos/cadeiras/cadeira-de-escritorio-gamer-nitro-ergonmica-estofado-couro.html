Visão Geral

A Cadeira Gamer Nitro LuvinCo foi projetada para oferecer máximo conforto durante longas sessões de jogos, trabalho ou estudo.

  • Assento com espuma de alta densidade.

Ergonomia para Longas Sessões

A cadeira foi desenvolvida para proporcionar postura adequada e conforto durante muitas horas de uso contínuo.

  • Apoio para pés retrátil para momentos de descanso.

Estrutura Robusta e Durável
